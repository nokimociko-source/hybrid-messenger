## Как использовать

### Для пользователей

1. Откройте любой чат
2. Нажмите на кнопку информации (ℹ️) в правом верхнем углу
3. Прокрутите вниз и найдите кнопку **"🔒 Шифрование E2E"**
4. В открывшемся окне включите переключатель
5. Нажмите "Сохранить"
6. Теперь все ваши сообщения будут шифроваться!

### Для разработчиков

#### Проверка включения E2E

```typescript
import { isE2EEnabled } from '../components/E2ESettings';

if (isE2EEnabled()) {
  console.log('E2E включено');
}
```

#### Использование хука

```typescript
import { useE2EEncryption } from '../hooks/useE2EEncryption';

const { isReady, error, encryptMessage, decryptMessage } = useE2EEncryption();

// Шифрование
const encrypted = await encryptMessage('Hello!', recipientUserId);

// Расшифровка
const decrypted = await decryptMessage(encrypted);
```

## Технические детали

### Хранение настроек

- **Ключ localStorage:** `e2e_enabled`
- **Значение:** `"true"` или `"false"`
- **Проверка:** `localStorage.getItem('e2e_enabled') === 'true'`

### Хранение ключей

- **Приватный ключ:** localStorage (`e2e_keys`)
- **Публичный ключ:** Supabase (`users.public_key`)
- **Формат:** Base64-encoded RSA keys

### Маркировка сообщений

Зашифрованные сообщения помечаются префиксом `🔒`:
```
🔒<encrypted_content>
```
