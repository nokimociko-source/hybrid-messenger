### 1. ✅ Исправлены проблемы с отображением пользователей и комнат

**Файлы**: `AdminSettings.tsx`

### 2. ✅ Обработка зашифрованных сообщений

**Проблема**: Зашифрованные сообщения отображались как нечитаемый JSON

**Решение**:
- Добавлена проверка на зашифрованное содержимое (начинается с `{` и содержит `ciphertext`)
- Зашифрованные сообщения отображаются как `[🔒 Зашифровано]` с оранжевым цветом
- Применено как в вкладке "Сообщения", так и в вкладке "Логи"

**Файлы**: `AdminSettings.tsx`

### 3. ✅ Система жалоб (Reports)

**Создано**:
- Таблица `reports` в базе данных с полями:
  - `id` - уникальный идентификатор
  - `reporter_id` - кто пожаловался
  - `reported_user_id` - на кого жалоба (опционально)
  - `reported_message_id` - на какое сообщение (опционально)
  - `reported_room_id` - на какую комнату (опционально)
  - `reason` - причина жалобы
  - `description` - описание (опционально)
  - `status` - статус (open, reviewing, resolved, rejected)
  - `admin_notes` - заметки администратора

**Компонент ReportModal**:
- Модальное окно для отправки жалоб
- Поддержка трех типов жалоб: на пользователя, на сообщение, на комнату
- Разные причины для каждого типа жалобы
- Валидация формы
- Обработка ошибок

**Вкладка "Жалобы" в админ-панели**:
- Просмотр всех жалоб
- Изменение статуса жалобы через dropdown
- Просмотр описания жалобы
- Автоматическое обновление при изменении статуса

**Файлы**:
- `database/migrations/add_reports_table.sql` - создание таблицы
- `database/migrations/add_admin_reports_access.sql` - доступ администраторов
- `client/src/app/components/ReportModal.tsx` - компонент модального окна
- `AdminSettings.tsx` - вкладка "Жалобы"
