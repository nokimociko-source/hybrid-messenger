    ├─ Видит: Публичные ключи
    ├─ Не видит: Приватные ключи
    └─ Результат: ❌ НЕ МОЖЕТ расшифровать

    ├─ Перехватывает: Зашифрованные сообщения
    ├─ Не знает: Приватные ключи
    ├─ Не знает: Эфемерные ключи
    └─ Результат: ❌ НЕ МОЖЕТ расшифровать

Атака 4: Компрометация одного ключа
    ├─ Скомпрометирован: Эфемерный ключ #5
    ├─ Расшифровано: Только сообщение #5
    ├─ Защищены: Все остальные сообщения (PFS)
    └─ Результат: ⚠️ Минимальный ущерб
