# 🎯 Боковая навигация в стиле Telegram

## Где находится?

Боковая панель навигации расположена **слева** на экране, шириной 72px.

## Структура

```
┌─────────────────────────────────────────┐
│  C   ← Логотип (вверху)                │
│                                         │
│  🏠  ← Главная (активна)               │
│  👤  ← Профиль                         │
│  🔍  ← Поиск                           │
│                                         │
│  ...  (пустое пространство)            │
│                                         │
│  ⚙️  ← Настройки (внизу)              │
└─────────────────────────────────────────┘
```

## Кнопки навигации

### 1. 🏠 Главная (Home)
- **Позиция**: Вверху, под логотипом
- **Функция**: Переход на главную страницу со списком чатов
- **Состояние**: Активна по умолчанию

### 2. 👤 Профиль (User)
- **Позиция**: Вторая сверху
- **Функция**: Открывает панель настроек профиля
- **Действие**: Показывает `ProfileSettingsPanel`

### 3. 🔍 Поиск (Search)
- **Позиция**: Третья сверху
- **Функция**: Открывает глобальный поиск
- **Действие**: Показывает `CatloverUserSearch`

### 4. ⚙️ Настройки (Settings)
- **Позиция**: Внизу панели
- **Функция**: Открывает настройки профиля
- **Действие**: Показывает `ProfileSettingsPanel`

## Как это работает?

### Интеграция в CatloverShell

```tsx
// hybrid_messenger/client/src/app/pages/client/CatloverShell.tsx

export function CatloverShell({ nav, children }: CatloverShellProps) {
    const [showProfileSettings, setShowProfileSettings] = useState(false);

    return (
        <div className={css.ShellRoot}>
            {/* Боковая панель */}
            <div className={css.TelegramBar}>
                {/* Иконки навигации */}
            </div>

            {/* Панель настроек профиля */}
            {showProfileSettings && (
                <div style={{ position: 'fixed', top: 0, left: '72px', zIndex: 1000 }}>
                    <ProfileSettingsPanel onClose={() => setShowProfileSettings(false)} />
                </div>
            )}
        </div>
    );
}
```

## Визуальный дизайн

### Размеры
- **Ширина панели**: 72px
- **Высота**: 100vh (на весь экран)
- **Размер иконок**: 48x48px
- **Отступы**: 16px сверху и снизу

### Цвета
- **Фон панели**: #0a0a0a
- **Граница**: rgba(0, 242, 255, 0.1)
- **Активная кнопка**: rgba(0, 242, 255, 0.15)
- **Hover**: rgba(255, 255, 255, 0.05)
- **Активный индикатор**: #00f2ff (полоска слева)

### Логотип
- **Размер**: 48x48px
- **Фон**: Градиент от #a200ff до #00f2ff
- **Текст**: "C" (первая буква Catlover)
- **Тень**: 0 4px 16px rgba(0, 242, 255, 0.3)

## Состояния кнопок

### Обычное состояние
```css
background: transparent
border: 1px solid transparent
color: rgba(255, 255, 255, 0.6)
```

### Hover
```css
background: rgba(255, 255, 255, 0.05)
```

### Активное
```css
background: rgba(0, 242, 255, 0.15)
border: 1px solid rgba(0, 242, 255, 0.3)
color: #00f2ff
+ индикатор слева (3px x 24px)
```

## Анимации

- **Transition**: all 0.2s
- **Hover эффект**: Плавное изменение фона
- **Активный индикатор**: Появляется слева от кнопки

## Открытие панели настроек

При клике на иконку профиля (👤) или настроек (⚙️):

1. Закрываются другие панели (поиск, старые настройки)
2. Открывается `ProfileSettingsPanel` справа от боковой панели
3. Панель позиционируется: `left: 72px` (сразу после боковой панели)
4. Z-index: 1000 (поверх всего)

## Закрытие панели

Панель закрывается:
- При клике на кнопку "Назад" (←) в панели
- При клике на другую кнопку навигации
- При клике вне панели (если добавить overlay)

## Пример использования

```tsx
// Открыть настройки профиля
<div onClick={() => setShowProfileSettings(true)}>
    <Icon src={Icons.User} />
</div>

// Панель настроек
{showProfileSettings && (
    <div style={{ position: 'fixed', top: 0, left: '72px', zIndex: 1000 }}>
        <ProfileSettingsPanel onClose={() => setShowProfileSettings(false)} />
    </div>
)}
```

## Адаптивность

На мобильных устройствах:
- Боковая панель может скрываться
- Кнопки могут быть в нижней панели
- Панель настроек открывается на весь экран
