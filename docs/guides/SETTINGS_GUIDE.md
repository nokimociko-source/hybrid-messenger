# 🎨 Руководство по настройкам приложения

## Где найти настройки?

Кнопка настроек находится в верхней панели списка чатов, рядом с кнопками поиска и создания чата.

**Иконка**: ⚙️ (шестеренка)

## Как открыть настройки?

1. Откройте приложение
2. В списке чатов найдите верхнюю панель с кнопками
3. Нажмите на кнопку с иконкой шестеренки (⚙️)
4. Откроется модальное окно с настройками

## Быстрый старт

### 1. Настройка профиля

**Путь**: Настройки → Профиль

- Нажмите на аватар для загрузки фото
- Обрежьте фото в удобном редакторе
- Введите имя пользователя
- Добавьте описание "О себе"
- Нажмите "Сохранить изменения"

### 2. Выбор темы

**Путь**: Настройки → Основные → Тема оформления

Доступны 3 варианта:
- 🌙 Темная (по умолчанию)
- ☀️ Светлая
- 🔄 Авто (следует системной теме)

### 3. Настройка уведомлений

**Путь**: Настройки → Уведомления

1. Включите мастер-переключатель уведомлений
2. Разрешите desktop notifications (если нужно)
3. Настройте звук и громкость
4. Выберите, показывать ли предпросмотр сообщений

### 4. Конфиденциальность

**Путь**: Настройки → Конфиденциальность

Настройте видимость:
- Статус "в сети"
- Время последнего визита
- Отчеты о прочтении
- Индикатор набора текста

### 5. Управление памятью

**Путь**: Настройки → Данные и память

- Просмотрите использование памяти
- Очистите кэш для освобождения места
- Настройте автоудаление старых сообщений
- Экспортируйте данные

## Полезные функции

### 🎨 Размер шрифта

**Путь**: Настройки → Основные → Размер шрифта

Регулируйте от 12px до 18px для комфортного чтения.

### 🔕 Отключение звука

**Путь**: Настройки → Уведомления → Звук уведомлений

Отключите звук или настройте громкость. Есть кнопка "Проверить звук".

### 📦 Очистка кэша

**Путь**: Настройки → Данные и память → Очистить кэш

Удаляет временные файлы, сохраняя важные данные.

### 🔧 Режим разработчика

**Путь**: Настройки → Расширенные → Режим разработчика

Включает дополнительные инструменты:
- Вывод данных хранилища
- Информация о системе
- Метрики производительности

## Горячие клавиши

В настройках пока нет горячих клавиш, но вы можете:
- **ESC** - закрыть настройки (нажмите на крестик)
- **← Назад** - вернуться в главное меню

## Сохранение настроек

Все настройки сохраняются автоматически:

- ✅ **Профиль** - сохраняется в Supabase при нажатии "Сохранить"
- ✅ **Остальные разделы** - сохраняются в localStorage при нажатии "Сохранить изменения"

## Сброс настроек

### Сброс отдельного раздела

В некоторых разделах есть кнопка "Сбросить" (например, в Основных настройках).

### Полный сброс

**Путь**: Настройки → Данные и память → Удалить все данные

⚠️ **ВНИМАНИЕ**: Это действие необратимо! Будут удалены:
- Все сообщения
- Все настройки
- Весь кэш
- Все данные приложения

## Экспорт данных

**Путь**: Настройки → Данные и память → Экспортировать данные

Сохраняет все ваши данные в файл для резервного копирования.

## Решение проблем

### Настройки не сохраняются

1. Проверьте, что нажали кнопку "Сохранить изменения"
2. Убедитесь, что localStorage не заблокирован в браузере
3. Проверьте консоль браузера на ошибки (F12)

### Не загружается аватар

1. Проверьте размер файла (должен быть < 5MB)
2. Используйте форматы: JPG, PNG, GIF
3. Проверьте подключение к интернету
4. Убедитесь, что Supabase Storage настроен

### Уведомления не работают

1. Проверьте разрешения браузера (Settings → Notifications)
2. Убедитесь, что уведомления включены в настройках приложения
3. Проверьте, не замьючен ли чат

## Для разработчиков

### Добавление нового раздела

1. Создайте файл `settings/YourSettings.tsx`
2. Добавьте в `menuItems` в `SettingsModal.tsx`
3. Добавьте case в `renderContent()`

### Добавление новой настройки

1. Добавьте поле в интерфейс `Preferences`
2. Добавьте в `defaultPreferences`
3. Создайте UI элемент (toggle, select, input)
4. Используйте `updatePreference()` для изменения

### Пример кода

```tsx
// Добавление toggle-переключателя
const renderToggle = (
    label: string,
    description: string,
    key: keyof Preferences,
    value: boolean
) => (
    <div onClick={() => updatePreference(key, !value)}>
        {/* UI */}
    </div>
);
```
