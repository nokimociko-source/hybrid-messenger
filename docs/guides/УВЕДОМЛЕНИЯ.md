# Система уведомлений - готова! 🔔

## Что работает

✅ **Уведомления о входящих звонках**
- Звук звонка (зацикленный)
- Системное уведомление с именем звонящего
- Вибрация на мобильных (когда добавим Capacitor)

✅ **Уведомления о новых сообщениях**
- Звук уведомления
- Системное уведомление с текстом сообщения
- Короткая вибрация на мобильных

✅ **Поддержка платформ**
- Браузер (Web) - работает
- Десктоп (Electron) - работает
- Мобильные (Capacitor) - готово к установке пакетов

## Как это работает

1. При запуске приложения запрашивается разрешение на уведомления
2. Когда приходит звонок - звучит `/sound/invite.ogg` + показывается уведомление
3. Когда приходит сообщение - звучит `/sound/notification.ogg` + показывается уведомление
4. Звук звонка автоматически останавливается при принятии/отклонении

## Для мобильной версии

Когда будем делать мобильное приложение, нужно установить:
```bash
npm install @capacitor/local-notifications @capacitor/haptics
```

## Файлы

- `src/app/utils/platformNotifications.ts` - основной код
- `electron/main.cjs` - обработчик для десктопа
- `electron/preload.cjs` - мост для IPC
- Интегрировано в хуки звонков и сообщений
