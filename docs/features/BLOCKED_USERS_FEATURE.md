# Функционал черного списка (блокировки пользователей)

## Обзор
Полноценная система блокировки пользователей с UI управлением и интеграцией в профили.

## Компоненты

### 1. BlockedUsersModal
Модальное окно для управления черным списком:
- Просмотр всех заблокированных пользователей
- Поиск и добавление новых пользователей в черный список
- Разблокировка пользователей
- Отображение даты блокировки

**Расположение**: `client/src/app/components/BlockedUsersModal.tsx`

### 2. PrivacySettings (обновлено)
Интеграция черного списка в настройки конфиденциальности:
- Кнопка "Заблокированные пользователи" с счетчиком
- Автоматическое обновление счетчика при изменениях
- Открытие BlockedUsersModal

**Расположение**: `client/src/app/components/settings/PrivacySettings.tsx`

### 3. UserProfileModal (обновлено)
Добавлена кнопка блокировки/разблокировки в профиле пользователя:
- Проверка статуса блокировки при открытии
- Кнопка "Заблокировать" / "Разблокировать"
- Подтверждение действия
- Автоматическое закрытие после блокировки

**Расположение**: `client/src/app/components/UserProfileModal.tsx`

## База данных

### Таблица blocked_users
```sql
CREATE TABLE blocked_users (
    id UUID PRIMARY KEY,
    user_id UUID NOT NULL,           -- Кто блокирует
    blocked_user_id UUID NOT NULL,   -- Кого блокируют
    blocked_at TIMESTAMP,             -- Когда заблокирован
    reason TEXT,                      -- Причина (опционально)
    UNIQUE(user_id, blocked_user_id)
);
```

### Функции
- `is_user_blocked(user_id, blocked_user_id)` - проверка блокировки
- `handle_user_blocked()` - триггер удаления чатов при блокировке

### RLS политики
- Пользователи видят только свой черный список
- Только владелец может добавлять/удалять записи

## Использование

### Открыть черный список
1. Настройки → Конфиденциальность
2. Нажать "Заблокированные пользователи"

### Заблокировать пользователя

**Способ 1: Через черный список**
1. Открыть черный список
2. Нажать "Добавить"
3. Ввести имя пользователя
4. Выбрать из результатов поиска
5. Подтвердить блокировку

**Способ 2: Через профиль**
1. Открыть профиль пользователя в чате
2. Нажать кнопку "Заблокировать"
3. Подтвердить действие

### Разблокировать пользователя

**Способ 1: Через черный список**
1. Открыть черный список
2. Найти пользователя
3. Нажать "Разблокировать"
4. Подтвердить действие

**Способ 2: Через профиль**
1. Открыть профиль заблокированного пользователя
2. Нажать "Разблокировать"
3. Подтвердить действие

## Что происходит при блокировке

✅ **Автоматически**:
- Удаляются все прямые чаты между пользователями
- Заблокированный не может отправлять сообщения
- Заблокированный не видит ваш онлайн-статус
- Вы не получаете уведомления от заблокированного

❌ **Не влияет на**:
- Групповые чаты (оба остаются участниками)
- Историю сообщений в группах
- Общие контакты

## API примеры

### Проверить блокировку
```typescript
const { data } = await supabase
    .rpc('is_user_blocked', {
        p_user_id: currentUserId,
        p_blocked_user_id: targetUserId
    });
```

### Заблокировать пользователя
```typescript
const { error } = await supabase
    .from('blocked_users')
    .insert({
        user_id: currentUserId,
        blocked_user_id: targetUserId
    });
```

### Разблокировать пользователя
```typescript
const { error } = await supabase
    .from('blocked_users')
    .delete()
    .eq('user_id', currentUserId)
    .eq('blocked_user_id', targetUserId);
```

### Получить список заблокированных
```typescript
const { data } = await supabase
    .from('blocked_users')
    .select(`
        id,
        blocked_user_id,
        blocked_at,
        user:users!blocked_users_blocked_user_id_fkey (
            id,
            username,
            avatar_url
        )
    `)
    .eq('user_id', currentUserId);
```
