## 🚀 Как протестировать

### Форматирование текста:
1. Открой любой чат
2. Нажми на кнопку "T" (форматирование) слева от поля ввода
3. Появится панель с кнопками B, I, </>, S, 🔗
4. Выдели текст и нажми кнопку - добавится разметка
5. Или просто пиши Markdown вручную: `**жирный**`, `*курсив*`, `` `код` ``
6. Отправь сообщение - оно отобразится с форматированием

### Slowmode:
1. Открой настройки группы (3 точки → Настройки)
2. Прокрути вниз до "Медленный режим (Slowmode)"
3. Выбери интервал (например, 30 секунд)
4. Сохрани
5. Теперь обычные участники могут писать только раз в 30 секунд
6. Админы могут писать без ограничений

### Светлая тема:
1. Открой настройки профиля (иконка шестерёнки)
2. Найди блок "Тема оформления"
3. Нажми на него
4. Тема переключится с тёмной на светлую (или наоборот)
5. Настройка сохранится автоматически
