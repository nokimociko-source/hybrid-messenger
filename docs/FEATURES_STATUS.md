## � ДЕТАЛЬНАЯ ПРОВЕРКА "НЕТ В БД/КОДЕ"

### ✅ ЕСТЬ В КОДЕ (но не используется полностью):
1. **Просмотры сообщений (read_by)** - ✅ ЕСТЬ
   - БД: `messages.read_by UUID[]` ✅
   - Хук: `useReadReceipts()` ✅
   - Функция: Массив ID пользователей кто прочитал ✅
   - UI: ❌ Нет отображения списка "кто прочитал"

2. **Переход к непрочитанному** - ⚠️ ЧАСТИЧНО ЕСТЬ
   - Код: `handleJumpToUnread()` в CatloverRoomView ✅
   - Состояние: `showJumpToUnread`, `firstUnreadMessageId` ✅
   - Кнопка: Есть в UI ✅
   - Проблема: Не находит правильное сообщение ⚠️

### ❌ РЕАЛЬНО НЕТ В БД/КОДЕ:

1. **Медленный режим (slowmode)**
   - БД: ❌ Нет таблицы/полей
   - Код: ❌ Нет функций
   - Нужно: `rooms.slowmode_interval`, проверка времени последнего сообщения

2. **Бан/мут участников**
   - БД: ❌ Нет таблицы banned_users/muted_users
   - Код: ❌ Нет функций
   - Примечание: `mute_settings` - это для отключения уведомлений, не для мута участников
   - Нужно: Новая таблица с banned_until, muted_until

3. **Изменение названия/аватара/описания группы**
   - БД: ✅ Поля есть (name, avatar_url, topic)
   - Код: ❌ Нет функций updateRoom()
   - UI: ❌ Нет формы редактирования
   - Нужно: Хук и UI для редактирования

4. **Публичные группы с username**
   - БД: ❌ Нет полей is_public, username
   - Код: ❌ Нет функций
   - Нужно: Поля в rooms, поиск по username

5. **Статистика группы**
   - БД: ❌ Нет таблицы room_stats
   - Код: ❌ Нет функций
   - Нужно: Сбор метрик (просмотры, активность, рост)

6. **Автоудаление сообщений**
   - БД: ❌ Нет поля delete_after/expires_at
   - Код: ❌ Нет функций
