# SettingsModal - Система настроек в стиле Telegram

Полноценная система настроек с красивым интерфейсом в стиле Telegram.

## Использование

```tsx
import { SettingsModal } from './SettingsModal';

function MyComponent() {
    const [isOpen, setIsOpen] = useState(false);

    return (
        <>
            <button onClick={() => setIsOpen(true)}>
                Настройки
            </button>

            {isOpen && (
                <SettingsModal 
                    onClose={() => setIsOpen(false)}
                    initialSection="main" // опционально
                />
            )}
        </>
    );
}
```

## Разделы настроек

### 1. 👤 Профиль (ProfileSettings)
- Загрузка и обрезка аватара
- Изменение имени пользователя
- Редактирование описания "О себе"
- Сохранение в Supabase

### 2. ⚙️ Основные (GeneralSettings)
- **Тема оформления**: Темная / Светлая / Авто
- **Язык интерфейса**: Русский / English
- **Размер шрифта**: 12-18px
- **Анимации**: Включить/выключить плавные переходы
- **Компактный режим**: Уменьшенные отступы

### 3. 🔔 Уведомления (NotificationSettings)
- **Мастер-переключатель**: Включить/выключить все уведомления
- **Desktop notifications**: Запрос разрешения браузера
- **Звук уведомлений**: Включить/выключить + регулировка громкости
- **Предпросмотр сообщений**: Показывать текст в уведомлении
- **Уведомления из групп**: Включить/выключить
- **Только упоминания**: Уведомлять только при @mention

### 4. 🔒 Конфиденциальность (PrivacySettings)
- **Статус "в сети"**: Показывать/скрывать
- **Время последнего визита**: Показывать/скрывать
- **Отчеты о прочтении**: Отправлять/не отправлять
- **Индикатор набора**: Показывать когда печатаете
- **Видимость фото профиля**: Все / Контакты / Никто
- **Приглашения в группы**: Все / Контакты / Никто
- **Звонки**: Все / Контакты / Никто
- **Заблокированные пользователи**: Управление списком

### 5. 💬 Чаты (ChatSettings)
- **Enter для отправки**: Включить/выключить
- **Показывать время**: Время отправки сообщений
- **Показывать аватары**: В списке сообщений
- **Показывать имена**: Имя отправителя
- **Группировка сообщений**: По отправителю / По времени / Не группировать
- **Размер эмодзи**: Маленький / Средний / Большой
- **Предпросмотр ссылок**: Показывать превью
- **Автозагрузка медиа**: Автоматически загружать файлы
- **Автовоспроизведение GIF**: Включить/выключить
- **Автовоспроизведение видео**: Включить/выключить
- **Наборы стикеров**: Управление стикерами

### 6. 💾 Данные и память (DataStorageSettings)
- **Использование памяти**: Визуализация занятого места
  - Сообщения
  - Медиафайлы
  - Кэш
- **Автоудаление**: Настройка автоматического удаления старых сообщений
- **Очистка кэша**: Удаление временных файлов
- **Экспорт данных**: Сохранение всех данных
- **Удаление всех данных**: Полная очистка (с подтверждением)

### 7. 🔧 Расширенные (AdvancedSettings)
- **Режим разработчика**: Дополнительные инструменты
- **Отладочные логи**: Вывод в консоль
- **Экспериментальные функции**: Новые возможности в тестировании
- **Аппаратное ускорение**: Использование GPU
- **Бета-обновления**: Ранние версии
- **Отчеты об ошибках**: Автоматическая отправка
- **Статистика использования**: Анонимная телеметрия
- **Инструменты разработчика**:
  - Вывод данных хранилища
  - Информация о системе
  - Метрики производительности
- **Информация о приложении**: Версия, сборка, платформа

## Хранение настроек

Все настройки сохраняются в `localStorage`:

- `general_preferences` - основные настройки
- `notification_preferences` - уведомления
- `privacy_preferences` - конфиденциальность
- `chat_preferences` - настройки чатов
- `advanced_preferences` - расширенные настройки

## Расширение

Чтобы добавить новый раздел:

1. Создайте компонент в `settings/YourSettings.tsx`
2. Добавьте в `menuItems` в `SettingsModal.tsx`
3. Добавьте case в `renderContent()`

Пример:

```tsx
// settings/YourSettings.tsx
export function YourSettings({ onBack }: { onBack: () => void }) {
    return (
        <div>
            {/* Ваши настройки */}
        </div>
    );
}

// SettingsModal.tsx
const menuItems = [
    // ...
    {
        id: 'your-section',
        icon: '🎨',
        label: 'Ваш раздел',
        description: 'Описание раздела',
    },
];
```
